# D.N. - Dev Notes

一個基於 Gatsby 建構的個人技術博客，專注於分享開發筆記和技術文章。

## 快速開始

### 安裝依賴
```bash
yarn install
```

### 開發模式
```bash
yarn start
```
瀏覽器將自動開啟 `http://localhost:8000`

### 建構專案
```bash
yarn build
```

### 本地預覽建構結果
```bash
yarn serve
```

## 部署

### 部署到 GitHub Pages
```bash
yarn deploy
```


## 技術棧

- **框架**: Gatsby.js
- **語言**: TypeScript
- **樣式**: Tailwind CSS
- **內容**: MDX/Markdown
- **部署**: GitHub Pages
